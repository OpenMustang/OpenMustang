# Contributing to OpenMustang

Thank you for your interest in contributing to OpenMustang!

## What you can do
- Suggest improvements
- Report issues
- Share test results
- Propose design changes
- Contribute CAD, documentation, or code

## Rules
- All contributions must be compatible with **CC BY-NC-SA 4.0**
- Commercial use of any contribution is not allowed
- Contributors must clearly document their changes
- No closed-source or proprietary additions

## How to contribute
1. Fork the repository
2. Create a feature branch
3. Commit clearly documented changes
4. Open a pull request

## Important
By contributing, you agree that your work will be released under  
**CC BY-NC-SA 4.0**.
