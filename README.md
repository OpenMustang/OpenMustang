# OpenMustang

OpenMustang is a DIY open-hardware project focused on building a realistic  
**P-51D Mustang cockpit for DCS World**.

The project follows a **modular architecture**
each cockpit module (stick, rudder, panels, instruments) is developed,
tested, and documented independently.

This project is intended for **non-commercial, educational, and personal use**.

---

## Project Goals
- Accurate DIY cockpit modules for DCS World
- Modular and scalable design
- DIY-friendly manufacturing (3D printing, basic CNC, PCB)
- Clear documentation and long-term maintainability
